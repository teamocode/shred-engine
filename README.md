### IntelliJ Setup
---
1. In the status bar, open `Run` -> `Edit Configurations..`
2. Click `Modify Options`, and ensure `Add VM options` is checked
3. Add `-XstartOnFirstThread` to the VM options
4. Click `Apply` and then `Ok`
